## ⚙️ Personalización

### Cambiar colores
En `styles.css`, busca `:root` y edita:
```css
--primary-500: #0B3C49;   /* Midnight Teal */
--gold-500: #C29E54;       /* Ámbar Dorado  */
```

### Cambiar textos
Todos los textos están en `index.html`, editables directamente.

### Cambiar imágenes
Reemplaza los archivos en `/images/` y mantén los mismos nombres.
